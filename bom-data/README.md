# bom-data

This directory contains data and shapefiles for the Death by Numbers project. 

- **data-csvs** contains data exports from the printed Bills of Mortality
- **deathdictionary.csv** contains definitions for causes of death found in the Bills of Mortality
- **geoJSON-files** contain geoJSON files for the parishes found within the Bills of Mortality, as they changed over time
- **London-parishes-authority-file.csv** contains metadata on the parishes found within the Bills of Mortality
- **parish-networks** contains network data for the parishes found within the Bills of Mortality, as they changed over time
- **parish-shapefiles** contains shapefiles for the parishes found within the Bills of Mortality, as they changed over time
- **monarchical-bills** contains data exports from the manuscript Bills of Mortality presented to the monarchs
